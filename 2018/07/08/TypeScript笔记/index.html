TypeScript笔记

发表于 2018-07-08 | 分类于 工具 | | 阅读次数

ES是客户端脚本语言的规范，ES5、ES6是这个规范的不同版本。
JavaScript，TypeScript是客户端是两种客户端脚本语言。
JavaScript实现了ES5规范，TypeScript实现了ES6规范。

一.TypeScript优点

1.支持ES6规范

2.强大的IDE支持

  • 类型检查

    为变量指定类型，IDE会做类型检查，告知错误

  • 语法提示

    在IDE里编写typeScript代码时，IDE会根据你所处的上下文提示你能用的类、变量、方法和关键字，选择即可。

  • 重构

    修改变量名、方法名、文件名时，IDE会自动修改引用该变量或调用方法的地方。

3.Angular2的开发语言

二.搭建TypeScript的开发环境

什么是compiler？

安装TypeScript的compiler（编译器），把TypeScript的代码转成JavaScript的代码。

为什么需要compiler？

因为ES6规范是2015年发布的，目前的浏览器还不支持直接运行ES6规范的代码。所以需要一个Compiler来转成JavaScript。

三.字符串新特性

1.多行字符串

2.字符串模板 ${}

3.自动拆分字符串

根据表达式做了切割，表达式以外的字符串被切割成数组放到方法的第一个参数template里。
第一个表达式放第二个参数，以此类推。

四.参数新特性

1.参数类型

  • 变量声明类型
  • 自定义类型

2.默认参数

3.可选参数

  • 在方法的参数声明后面用问号来标明此参数为可选参数
  • 需处理可选参未传时的的情况
  • 可选参需声明在必选参后

五.函数特性

1.Rest and Spread 操作符

  • 用来声明任意数量的方法参数
  • …可以传任意数量参数，…声明的args参数

2.generator函数

  • 控制函数的执行过程，手工暂停和恢复代码执行(ES6规范，ts暂不支持）
  • 通过*声明一个generator函数，在函数中通过yield关键字控制代码的暂停和继续进行

3.Destructing析构表达式

通过表达式将对象或数组拆解成任意数量的变量

  • 通过析构表达式从对象里拆一些值出来声明本地变量
  • 通过析构表达式从数组拆一些值出来声明本地变量

六.表达式与循环

1.箭头表达式

用来声明匿名函数，消除传统函数的this指针问题

2.forEach(),for in和for of

七.面向对象特性

1.类

类是TypeScript的核心，使用TypeScript开发时，大部分代码都是写在类里面的。
这里介绍类的定义，构造函数，以及类的继承

  • 类的定义

    在javaScript里类的继承都是通过prototype关键字实现的

  • 类的实例化
  • 类的访问控制符(3个)

    1.public 默认访问控制符，可在类的内部、外部都被访问
    2.private 私有的，只能在类的内部被访问，外部不可
    3.protected 受保护的，可在类的内部与子类被访问，外部不可

  • 类的构造函数-constructor
  • 类的继承

涉及到连个关键字：
extends 用来声明类的继承关系，当一个类继承了另一个类，这个类就获得他所继承的类的所有属性和方法 super 调用父类的构造函数<子类的构造函数必须要调用父类的构造函数>或者方法<super.方法>

2.泛型

参数化的类型，一般用来限制集合的内容

3.接口（Interface）

用来建立某种代码约定，使得其它开发者在调用某个方法或创建新的类时必须遵循接口所定义的代码约定。在javascript中没有接口这个概念。
在typescript中，提供两个关键字支撑接口这个特性，interface(声明结构)和implemen(用于声明某个类实现了某个接口)

4.模块（Module）

模块可以帮助开发者将代码分割为可重用的单元。开发者可以自己决定模块中哪些资源（类、方法、变量）暴露出去供外部使用，哪些资源只在模块内使用。
typeScript里一个文件就是一个模块。

  • export : 导出，对外暴露
  • import : 导入，引入外部资源

5.注解（annotation）

注解为程序的元素（类、方法、变量）加上更直观更明了的说明，这些说明信息与程序的业务逻辑无关，而是供指定的工具或者框架使用的。
告诉框架如何处理程序的元素。

6.类型定义文件（*.d.ts）

类型定义文件用来帮助开发者在TypeScript中使用已有的JavaScript的工具包。
有几乎所有常用的javaScript框架的类型定义文件的网址：https://github.com/DefinitelyTyped/DefinitelyTyped。

github搜typings工具，用来安装类型定义文件
